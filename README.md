# Photobooth Web App

## Overview
This is a simple, stylish web‑based photobooth that works in any modern browser. It accesses the webcam, lets the user capture a picture, and provides an instant **Download** button to save the image as `photo.png`.

## Files
- `index.html` – The main page with markup and JavaScript.
- `style.css` – Modern glass‑morphism styling with dark gradient and smooth animations.

## Running
1. Open `index.html` in a browser (Chrome, Edge, Firefox, etc.).
2. The browser will ask for permission to use the camera – **Allow** it.
3. Click the **Capture** button to take a photo.
4. The **Download** button appears; click it to save the picture.

## Notes
- Works best on desktop/laptop browsers with a webcam.
- No server is required – it runs completely client‑side.
- The app is responsive and adapts to mobile screens.
